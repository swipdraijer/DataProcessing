A sequential color series is used to logically arrange the data from high values to low values based on lightness. This way low numbers are represented by light shades of blue and high numbers by dark shades of blue. As the light-to-dark progression is data-dependent this is inituitive and understandable without comparing each shade to the legend. The bounds of encoding in each category are based on dynamic binning of numbers: the range of values between the minimum and maximum value in the data set is divided by the number of bins, which equals the number of colors to be used.

Interactivity:

  • Highlight borders and fill country to put focus on country of interest
  • Popup window to add detailed information for each country
  • Visual queries in popup window to show raw values for life expectancy and to show in color red/green whether this value is below or above the global average. Visual queries map does not support: to click on country or popup to gain additional information as this does not greatly improve the visualization's message.
